You will be able to export the Scenarios configured in the valQ through
an URL by entering the Writeback URL in the Text Box (see Figure below).

<img alt="Export Scenarios via Writeback URL" src={useBaseUrl('/doc-images/wrb.png')} />

The different simulations of planning results can be written back to the
underlying systems using the URL exposed as part of the valQ. 

Consider the Demand and Inventory nodes to follow the trend as shown below.

| Parameter          | HY 1   | HY 2   |
|--------------------|--------|--------|
| Demand             | 110    | 95     |
| Inventory          | 85     | 70     |
| Demand - Inventory | 25     | 25     |


<img alt="Export Scenarios via Writeback URL" src={useBaseUrl('/doc-images/writeback.png')} />

The user can do write-back scenario nodes to the underlying 
system by clicking on the writeback icon in the navigation panel.

For the above tree, on writing back to the URL, the below data will be posted 
to the URL which can be processed as needed.
